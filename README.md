Симуляция маятника Ньютона, созданная в Unity с использованием собственного математического движка маятника и аналитического разрешения столкновений.

Маятник Ньютона - классический демонстрационный прибор, состоящий из пяти подвешенных металлических шаров. При отклонении одного или нескольких шаров с одной стороны и их последующем падении происходит удар о неподвижные шары, после чего равное количество шаров с противоположной стороны отлетает - демонстрируя законы сохранения импульса и энергии.

В отличие от стандартного подхода Unity (Rigidbody + HingeJoint), данная симуляция использует собственный движок маятника с математически точной обработкой столкновений, что полностью исключает потери энергии, характерные для встроенного физического решателя Unity.

Возможности
- Кастомная симуляция маятника на основе уравнения движения: `α = -(g/L) · sin(θ)`
- Аналитически точное разрешение упругих столкновений (обмен импульсами равных масс)
- Поддержка запуска 1–4 шаров одновременно
- Работа через префаб — достаточно перетащить в любую сцену
- Автоматическое определение рендер-пайплайна (Built-in / URP / HDRP)
- Настраиваемые параметры: количество шаров, радиус, длина нити, затухание, угол отклонения

Принцип работы

Физический движок
Каждый шар описывается двумя величинами: угол `θ` и угловая скорость `ω`. На каждом шаге физики:
1. Вычисляется угловое ускорение: `α = -(g/L) · sin(θ)`
2. Угловая скорость интегрируется и затухает
3. Позиция шара определяется из угла
Разрешение столкновений

Столкновения разрешаются аналитически за 8 итераций решателя за кадр:
- Шар, находящийся в контакте с соседним (`расстояние ≤ диаметр + ε`) и движущийся навстречу, мгновенно отдаёт свою угловую скорость (абсолютно упругий удар равных масс)
- Перекрытие устраняется путём корректировки углов

Такой подход гарантирует, что волна импульса проходит через все неподвижные шары за один физический кадр — воспроизводя характерный эффект маятника Ньютона.
